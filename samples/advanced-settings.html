Advanced Settings: Complete Guide

Category: Product Documentation | Last Updated: December 2024

Advanced Settings provide power users and administrators with granular control over their CloudSync experience. This guide covers all advanced configuration options, their use cases, and recommended settings for different scenarios.

Accessing Advanced Settings

To access Advanced Settings:

  1. Click your profile icon in the top-right corner
  2. Select "Settings" from the dropdown menu
  3. Navigate to the "Advanced" tab

Note: Some advanced settings require administrator privileges or specific plan levels.

Sync Configuration

Selective Sync

Selective Sync allows you to choose which folders sync to your local device, saving disk space and bandwidth.

  • Enable Selective Sync: Toggle this option to enable folder selection
  • Choose Folders: Select specific folders to sync locally
  • Smart Sync: Automatically sync recently accessed files while keeping others cloud-only
Use Case: If you have 500GB of files but only 100GB of local storage, use Selective Sync to keep only frequently accessed files on your device.

Bandwidth Throttling

Control how much bandwidth CloudSync uses for syncing:

  • Unlimited: Use all available bandwidth (default)
  • Custom Limit: Set a maximum upload/download speed in Mbps
  • Schedule: Set different limits for specific times of day

Recommended: Set a limit of 80% of your total bandwidth to leave room for other applications.

Conflict Resolution

Configure how CloudSync handles file conflicts when the same file is edited on multiple devices:

  • Ask Me: Prompt you to choose which version to keep (recommended)
  • Keep Newer: Automatically keep the most recently modified version
  • Keep Both: Save both versions with different names

Security Settings

Two-Factor Authentication (2FA)

Enhance your account security with 2FA:

  1. Navigate to Security → Two-Factor Authentication
  2. Choose your preferred method:
    • Authenticator App: Use apps like Google Authenticator or Authy
    • SMS: Receive codes via text message
    • Hardware Key: Use a physical security key (YubiKey, etc.)
  3. Follow the setup wizard to complete configuration
  4. Save backup codes in a secure location

Session Management

View and manage active sessions across all your devices:

  • Active Sessions: See all devices currently signed in
  • Device Names: Label devices for easy identification
  • Remote Sign Out: Sign out from specific devices
  • Session Timeout: Automatically sign out inactive sessions after a set period

IP Whitelisting

Restrict account access to specific IP addresses (Enterprise plans only):

  1. Enable IP Whitelisting in Security settings
  2. Add allowed IP addresses or IP ranges
  3. Configure exceptions for mobile devices (if needed)

Warning: If you enable IP whitelisting and your IP changes, you may be locked out. Ensure you have a backup access method.

Sharing and Collaboration

Link Sharing Permissions

Control default permissions for shared links:

  • View Only: Recipients can view but not download
  • Download: Recipients can view and download
  • Edit: Recipients can modify files (Professional/Enterprise)
  • Password Protection: Require a password for all shared links
  • Expiration: Set default expiration dates for shared links

Team Folder Permissions

Configure default permissions for team folders (Professional/Enterprise):

  • Admin Access: Full control over folder settings
  • Editor Access: Can add, edit, and delete files
  • Viewer Access: Can only view and download files
  • Custom Roles: Create custom permission sets for specific needs

Automation and Workflows

Automated Backups

Set up automatic backups for important folders:

  1. Navigate to Automation → Backups
  2. Click "Create Backup Rule"
  3. Select source folder and destination
  4. Choose backup frequency:
    • Real-time (immediate sync)
    • Daily (at specified time)
    • Weekly (on specific day)
    • Monthly (on specific date)
  5. Configure retention policy (how long to keep backups)

File Versioning

Control how file versions are managed:

  • Version History: Enable to keep previous versions of files
  • Retention Period: How long to keep old versions (30, 90, 365 days, or unlimited)
  • Auto-Cleanup: Automatically delete versions older than retention period
  • Version Limit: Maximum number of versions per file

Performance Optimization

Cache Settings

Optimize local cache for better performance:

  • Cache Size: Set maximum cache size (default: 1GB)
  • Cache Location: Choose where cache files are stored
  • Auto-Cleanup: Automatically clear cache when it reaches size limit

Indexing Options

Configure file indexing for faster search:

  • Full-Text Search: Index file contents for content search
  • Index File Types: Choose which file types to index
  • Background Indexing: Index files when system is idle

Integration Settings

Third-Party Integrations

Connect CloudSync with other services:

  • Email Integration: Save email attachments directly to CloudSync
  • Calendar Sync: Sync calendar events and attachments
  • Slack Integration: Share files directly from Slack
  • API Access: Generate API keys for custom integrations

Notification Preferences

Customize what notifications you receive:

  • File Updates: Notify when files are modified
  • Shared Links: Notify when someone accesses your shared links
  • Team Activity: Notify about team folder changes
  • Security Alerts: Notify about login attempts and security events
  • Email Digest: Receive daily or weekly summaries

Data Management

Export Data

Export all your data for backup or migration:

  1. Go to Data Management → Export
  2. Select data types to export (files, settings, activity logs)
  3. Choose export format (ZIP, JSON, or CSV)
  4. Click "Start Export" and wait for email notification
  5. Download export file from the provided link

Data Retention

Configure how long deleted files are kept:

  • Trash Retention: 30, 60, or 90 days (default: 30)
  • Permanent Deletion: Files are permanently deleted after retention period
  • Extended Retention: Keep deleted files longer (Enterprise feature)

Recommended Settings by Use Case

For Individual Users

  • Enable Smart Sync to save local storage
  • Set conflict resolution to "Ask Me"
  • Enable 2FA for security
  • Configure automated backups for important folders

For Small Teams

  • Set up team folders with appropriate permissions
  • Configure link sharing defaults
  • Enable file versioning with 90-day retention
  • Set up email integration for seamless workflow

For Enterprise

  • Implement IP whitelisting for enhanced security
  • Create custom roles and permission sets
  • Configure extended data retention policies
  • Set up comprehensive audit logging
  • Enable SSO (Single Sign-On) integration
